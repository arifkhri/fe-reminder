# FE HR App

 FE HR App is frontend web using react framework

---
## Requirement

### Node

    $ node --version
    v14.18.1

    $ npm --version
    6.14.15
    
## Installation

    $ git clone https://github.com/arifkhri/fe-hr-app/
    $ cd fe-hr-app
    $ npm i

## Running the project

    $ npm run start
    $ npm run build
