# FE Reminder

A project to exercise internship student. This project is CSR based using react JS, rest-api (axios) and antd

---
## Requirement

### Node

    $ node --version
    v14.18.1

    $ npm --version
    6.14.15
    
## Installation

    $ git clone https://github.com/arifkhri/fe-reminder/
    $ cd fe-reminder
    $ npm i

## Running the project

    $ npm run start
    $ npm run build
